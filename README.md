# Sistema de Tienda de Abarrotes

Aplicación desarrollada para la gestión de productos y ventas en una tienda de abarrotes, implementando las operaciones CRUD.

## Funcionalidades
- Registro y control de productos
- Gestión básica de ventas
- Manejo de información local

## Tecnologías utilizadas
- PHP, MySQL, HTML, CSS, JavaScript.

## Estado del proyecto
Proyecto mayormente funcional con algunas mejoras pendientes.
